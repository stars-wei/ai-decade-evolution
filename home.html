Before 2016

十年前，地基已经打好

这一阶段的关键不是某个单点，而是 GPU、深度神经网络、大数据、开源框架和研究社区汇合，给 2016 之后的爆发准备了土壤。

1950s-1980s

符号主义与专家系统

可解释、规则明确，但维护成本高，开放世界泛化差。

1997-05-11

Deep Blue

搜索与专用评估函数在封闭博弈中达到公众里程碑。

2012-09

AlexNet

ImageNet 突破让 GPU + CNN 成为视觉 AI 主干。

2012-2016

CNN 黄金期

VGG、Inception、ResNet 推动分类、检测、分割、OCR 与医学影像落地。

2013-2014

word2vec / seq2seq / GAN

表示学习、序列建模、生成式视觉各自点火。

2014-2020

GAN 高光期

DCGAN、CycleGAN、pix2pix、StyleGAN 推动图像生成、风格迁移、人脸合成与超分辨率。

2014-2017

RNN / LSTM 黄金期

seq2seq 与 Attention 推动机器翻译、语音识别、摘要、时间序列和早期对话系统。

2015-12-10

何恺明 / ResNet

残差连接让百层级 CNN 可训练，CVPR 2016 获 Best Paper，影响几乎所有现代深层网络。

长期积累

Hinton / Bengio / LeCun

深度学习路线重新成为主流，并在产业里找到了入口。

2016
深度强化学习的公众时刻

AlphaGo 与框架竞争

  • 2016-03-09 至 03-15，AlphaGo 4:1 战胜李世石，深度学习 + 搜索 + 强化学习证明复杂决策能力。
  • TensorFlow、Caffe、Theano、Torch、MXNet、CNTK 等框架竞争，TensorFlow 与 PyTorch 路线逐渐胜出。
  • 中国视觉 AI 公司在安防、金融、手机视觉中扩张，但后续受隐私、监管和商业化约束热度下降。
2017
Transformer 出现

自注意力成为新地基

  • 2017-06-12，Attention Is All You Need 发布，Transformer 成为后续 GPT、BERT、T5、ViT 和多模态模型核心。
  • 2017-07-20，中国发布《新一代人工智能发展规划》，AI 上升为国家战略。
  • AlphaGo Zero 展示自我博弈学习，但强化学习走出棋类后面临样本效率和现实成本瓶颈。
2018
预训练 + 微调

NLP 主范式切换

  • 2018-06，OpenAI 发布 GPT-1，验证 Transformer 解码器的无监督预训练迁移能力。
  • 2018-10-11，Google 发布 BERT，双向 Transformer + MLM 将 NLP 带入预训练时代。
  • AutoML/NAS 热度上升，但没有成为大模型时代主路线。
2019
发布治理出现

GPT-2 与 Hugging Face 生态

  • 2019-02-14，OpenAI 发布 GPT-2 并延迟完整权重，模型能力与滥用风险进入公共讨论。
  • 2019-11，完整 GPT-2 发布。开放发布与安全延迟发布的张力成为前沿模型常态。
  • Hugging Face Transformers 成长，模型复用、分享和微调门槛降低。
2020
规模定律

GPT-3 与 AlphaFold2

  • 2020-05-28，GPT-3 论文发布，175B 参数和少样本能力让“提示词”成为交互方式。
  • Scaling Laws 将更多数据、参数、计算变成工程路线，带动 GPU、集群训练和数据中心产业链。
  • 2020-11，AlphaFold2 在 CASP14 取得突破，AI for Science 从示范走向实用。
2021
多模态与代码

CLIP、DALL-E、Copilot、AlphaFold

  • 2021-01-05，OpenAI 发布 CLIP 和 DALL-E，文本-图像对齐与文本生图进入主线。
  • 2021-06-29，GitHub Copilot 技术预览发布，AI 编程进入 IDE 工作流。
  • 2021-07-15，AlphaFold2 论文发表于 Nature；07-22，AlphaFold 蛋白结构数据库发布。
2022
生成式 AI 爆发

扩散模型与 ChatGPT

  • 2022-04，DALL-E 2 发布，图像生成质量与可控性提升。
  • 2022-07-12，Midjourney 开放 beta；2022-08-22，Stable Diffusion 发布，开放图像生成生态爆发。
  • 2022-11-30，ChatGPT 发布，聊天界面 + 指令微调 + RLHF 让大模型成为大众工具。
2023
平台化与开源权重

GPT-4、Llama、Claude、Gemini

  • 2023-02-24，Meta 发布 LLaMA，催生本地模型、量化、llama.cpp、Alpaca/Vicuna 等生态。
  • 2023-03-14，GPT-4 发布；03-16，百度发布文心一言；04 月阿里通义千问/Qwen beta。
  • 2023-07-11，Claude 2 发布；2023-12-06，Google Gemini 1.0 发布。
  • 2023-11，OpenAI 董事会风波显示前沿 AI 公司治理与商业化的张力。
2024
实时多模态与法规

GPT-4o、Claude 3、Llama 3、EU AI Act

  • 2024-02，Gemini 1.5 推出百万级上下文；OpenAI 预览 Sora。
  • 2024-03-04，Claude 3 系列发布；03-13，欧洲议会通过 EU AI Act。
  • 2024-04-18，Llama 3 发布；05-13，GPT-4o 发布，实时文本、视觉、语音交互成为主叙事。
  • 2024-12-09，Sora 向付费用户发布，视频生成赛道进入拥挤竞争。
2025
推理模型与 Agent

DeepSeek-R1、Operator、Deep Research、GPT-5

  • 2025-01-20，DeepSeek-R1 发布，低成本开源推理模型冲击全球市场。
  • 2025-01-23，OpenAI Operator 研究预览；02-02，Deep Research 发布。
  • 2025-02-24，Claude 3.7 Sonnet 与 Claude Code 发布；03-25，Gemini 2.5 Pro Experimental 强化 thinking model 路线。
  • 2025-08-07，OpenAI 发布 GPT-5，统一快速回答与深度思考模型路线。
2026
模型变成系统

从聊天到工作流、代理与科学

  • 2026-02-12，Google 发布 Gemini 3 Deep Think，面向科学、工程与复杂推理。
  • 2026-04，Anthropic 继续强化 Claude 家族的 coding、agent 与 safety。
  • 2026-04-23，OpenAI 发布 GPT-5.5；04-24 API 可用，强调编码、计算机使用、研究和复杂文档工作。
  • 2026-05-11，MIT 等研究者发布 ELF: Embedded Language Flows，在连续嵌入空间中用 Flow Matching 生成语言，只在最后一步映射回 token。
  • 2026-05，ByteDance Seed 开源 Cola DLM，探索连续潜空间扩散语言模型；它与 ELF 一起构成对自回归 LLM 的研究性挑战。
  • 截至 2026-05-26，竞争焦点转向模型路由、工具调用、浏览器/电脑使用、长上下文、代码代理、企业知识库与安全评测。
Branches

有的长成森林，有的退回边缘

十年演化不是单线前进，而是多条路线彼此吸收、替代和重组。

Transformer 壮大

从 NLP 架构变成通用序列、多模态、代理系统底座。

基础模型 壮大

BERT/GPT 之后，LLM、VLM、音视频模型走向统一多模态。

扩散模型 壮大

从图像生成扩展到视频、3D、音频、编辑和设计工具。

AI for Code 壮大

从自动补全走向代码库理解、修 bug、测试、重构与长时 agent。

AI for Science 壮大

AlphaFold 带动蛋白、材料、药物、天气、数学和实验设计。

ELF / Embedded Language Flows 萌芽

何恺明等提出的连续嵌入空间语言流模型，用 Flow Matching 生成文本，只在最终步骤离散化；截至 2026 年仍是研究分支。

开放权重生态 壮大

Llama、Mistral、Qwen、DeepSeek、Kimi 等形成闭源 API 的平行路线。

DLM / 扩散语言模型 萌芽

ELF 与 Cola DLM 分别沿连续嵌入空间、连续潜变量空间探索非自回归文本生成；截至 2026 年仍是研究分支。

专家系统 退潮

可解释但脆弱，维护成本高，难适应开放世界。

CNN / 卷积网络 被吸收

从视觉 AI 的绝对主干，变成与 ViT、多模态模型并存的高效工程路线；在移动端、边缘视觉、检测分割和医学影像中仍有生命力。

RNN/LSTM 主干 边缘化

曾是文本、语音和时间序列建模核心；Transformer 之后退到流式语音、边缘设备、低延迟预测和部分控制系统。

GAN 通用生图 退潮

曾代表生成式视觉前沿，StyleGAN 带来高质量人脸和潜空间编辑；扩散模型崛起后退到实时生成、风格迁移、超分辨率、图像翻译和数据增强。

早期 AutoGPT 重构

概念重要，但无约束循环代理不稳定，后来被权限、工作流、评测和沙箱吸收。

Actors

公司、人物与产品版图

前沿 AI 是论文、产品、算力、资本、治理和用户反馈共同塑形的结果。

OpenAI

Sam Altman、Greg Brockman、Ilya Sutskever、Mira Murati 等；GPT、DALL-E、Whisper、ChatGPT、Sora、o 系列、Operator、Deep Research、GPT-5/5.5。

Google DeepMind

Demis Hassabis、John Jumper 等；AlphaGo、AlphaFold、Gemini、AI for Science，以及 Search、Android、Workspace 集成。

Anthropic

Dario Amodei、Daniela Amodei、Jared Kaplan 等；Claude、Constitutional AI、长上下文、安全分级、Claude Code。

Meta AI

Yann LeCun、Joelle Pineau、FAIR；Llama 开放权重生态成为最大产业影响。

何恺明

ResNet 主要作者之一，残差学习让超深 CNN 训练成为可行路线，并影响检测、分割、生成模型和 Transformer 残差结构。

NVIDIA

Jensen Huang；CUDA、H100/B 系列、网络、推理栈和数据中心构成算力底座。

中国 AI 公司

百度文心、阿里 Qwen、腾讯混元、字节豆包、智谱 GLM、月之暗面 Kimi、DeepSeek 等。

从“专用模型解决专用任务”，走向“通用基础模型 + 多模态输入输出 + 工具调用 + 长上下文/记忆 + 可执行代理 + 行业工作流”。
  1. 2016-2017：深度学习证明复杂决策能力，Transformer 出现。
  2. 2018-2020：预训练、规模定律、少样本学习形成语言模型主线。
  3. 2021-2022：多模态、扩散模型、代码模型和 ChatGPT 把 AI 推向大众。
  4. 2023-2024：闭源前沿模型与开放权重模型双线竞争，监管启动。
  5. 2025-2026：推理模型和 agent 化系统成为核心，竞争转向复杂工作完成能力。
