STAGE 01 (2016 - 2020)

深度强化学习与基石时代

这一时期奠定了现代大模型的基础。以深度强化学习在游戏博弈中的突破为起点，引出了统治下一个十年的 Transformer 底层架构，同时 PyTorch 开源，预训练与微调的双轨范式初步建立。

2016-03-09 公司/事件

AlphaGo 击败围棋世界冠军李世石

Google DeepMind 的 AlphaGo 在韩国首尔与李世石展开五番棋对决，最终以 4:1 获胜。这是人工智能史上的破圈时刻，向公众展现了深度强化学习在极其复杂的博弈空间中超越人类顶尖智力的能力。

2016-04-27 框架/工具

OpenAI Gym 平台发布

OpenAI 推出 Gym 开源套件，用于开发和对比强化学习算法。它迅速演变成为全球强化学习学术研究和工业测试的标准环境。

2016-06-20 技术/算法

ResNet 斩获 CVPR 最佳论文

何恺明等人发表的《Deep Residual Learning for Image Recognition》解决深层网络梯度消失问题。残差连接（Residual Connection）直接成为后来计算机视觉和 Transformer 架构不可或缺的底层支柱。

2016-10-04 框架/工具

PyTorch 框架正式开源

Facebook (现 Meta) 推出 PyTorch，以其动态计算图（Dynamic Graph）和 Pythonic 设计迅速赢得学术界青睐，拉开了与 Google TensorFlow 的学术市场争夺战。

2017-06-12 技术/算法

《Attention Is All You Need》提出 Transformer 架构

Google 团队发表论文，完全放弃 RNN 架构，提出基于自注意力机制（Self-Attention）的 Transformer。该架构拥有极高的并行化计算能力与卓越的长短序列建模能力，成为现代所有 LLM 的骨架。

2017-10-18 模型

AlphaGo Zero 问世：告别预训练人类数据

DeepMind 发布 AlphaGo Zero，完全通过自我博弈强化学习，不依赖任何人类经验。在训练仅 3 天后便以 100:0 零封前代 AlphaGo，证明了自监督强化学习在博弈问题上的无限可能。

2018-06-11 模型

OpenAI GPT-1 发布

OpenAI 推出首个生成式预训练 Transformer (GPT-1)，确立了“单向 Transformer 解码器 + 自监督在大规模文本预训练 + 下游微调”的演化路径。

2018-10-11 模型

Google BERT 霸榜自然语言任务

Google 推出 BERT 模型（双向编码器结构），在 11 项核心 NLP 任务上刷新世界纪录。BERT 成为自然语言理解的工业标配，与 OpenAI 的 GPT 自回归路线并存竞合。

2019-02-14 公司/事件

OpenAI 宣布 GPT-2 且延迟开源引发安全辩论

OpenAI 推出拥有 15 亿参数的 GPT-2 模型，因其生成文本过于逼真，官方以“安全风险”为由暂缓开源，引发学术界关于安全偏见与科学透明的激辩。

2019-07-22 公司/事件

微软向 OpenAI 注资 10 亿美元

微软对 OpenAI 进行 10 亿美元战略投资，双方达成深度云服务排他性合作。这笔巨资打通了 OpenAI 走向超级大模型开发所需的超算通道。

2020-05-28 模型

GPT-3 发布：确立 Scaling Law 与上下文学习

OpenAI 推出 1750 亿参数的 GPT-3。由于参数规模的指数级增加，模型首次表现出无需微调、仅需提供 Prompt 的 **In-Context Learning（上下文学习）** 的能力，彻底揭开了参数即智能的暴力美学时代。

2020-11-30 模型

AlphaFold 2 解决生物界蛋白质折叠之谜

在 CASP14 竞赛中，DeepMind 研发的 AlphaFold 2 在准确性上达到了原子级精度，完美破解了困扰生物学界 50 年的难题，标志着 AI for Science（科学人工智能）进入全新篇章。

STAGE 02 (2021 - 2023)

生成式 AI 与大爆发时代

这一时期见证了生成式 AI 从实验室走向大众的狂欢。DALL-E 2 与 Stable Diffusion 彻底颠覆了视觉生成技术，而 ChatGPT 的降临彻底将大语言模型融入到各行各业的生产力中，掀起了史诗级的巨头混战与开源反超潮。

2021-01-05 技术/算法

OpenAI 推出 CLIP：打通图像与文本特征空间

OpenAI 发布 CLIP，通过对比学习将图像和文本映射到同一向量空间。它成为后来所有文生图（AI Art）模型的核心检索与导航引擎。

2021-06-29 框架/工具

GitHub Copilot 开发者工具亮相

GitHub 联合 OpenAI 发布 Copilot，基于大规模开源代码库微调的 OpenAI Codex。AI 编程助手第一次走入程序员开发流，大幅提升编码效率。

2022-01-27 技术/算法

InstructGPT 开启 RLHF（人类反馈强化学习）时代

OpenAI 推出 InstructGPT，通过基于人类反馈的强化学习来对齐（Alignment）模型意图。大模型开始能够温顺听懂复杂人类指令，并减少恶意和虚假输出。

2022-08-22 框架/工具

Stable Diffusion 开源：生成艺术的平权运动

Stability AI 联合开源了基于潜在扩散模型（Latent Diffusion）的 Stable Diffusion，普通显卡即可跑出大师级艺术画作，瞬间点燃全球 AIGC 社区，也直接拉开了与版权方的长期法律较量。

2022-11-15 公司/事件

Meta Galactica 模型上线 3 天因虚假学术内容下线

Meta 推出科学计算语言模型 Galactica，因其生成的内容经常听起来专业却充满学术错误（幻觉），面临学术界一致声讨，被迫在发布第 3 天紧急撤下。

2022-11-30 模型

ChatGPT 横空出世，AI 历史的 iPhone 时刻

OpenAI 推出基于 GPT-3.5 的对话界面应用 ChatGPT，通过易用的聊天界面和人性化的反馈直接征服世界。产品在两个月内活跃用户突破 1 亿，成为互联网发展史上增长最快的消费级应用，开启了生成式 AI 新浪潮。

2023-02-24 框架/工具

Meta LLaMA 开源：点燃大模型开源燎原火种

Meta 推出 LLaMA-1 并在学术界开源，随后权重在网上被完全泄露。全球研究者在此基础上开展了低成本微调（如 Alpaca, Vicuna），使得大模型不再为少数巨头所独占，开源模型生态自此大爆发。

2023-03-14 模型

GPT-4 正式发布：逻辑推理与多模态的突破

OpenAI 推出 GPT-4，表现出强大的逻辑推理、多步骤规划和图像识别分析能力，并在人类多项高难度考试中斩获前 10% 的拔尖成绩，树立了闭源商业模型的性能标杆。

2023-11-17 公司/事件

OpenAI “11.17” 董事会大地震

OpenAI 董事会宣布罢免 CEO Sam Altman，引发全公司超 90% 员工联合签名离职抗意与微软的深度博弈。在经历极其戏剧化的 5 天反转后，Altman 携新董事会复职，首席科学家 Ilya Sutskever 边缘化并在数月后出走。

STAGE 03 (2024 - 2026)

智能体化、深度推理与效率革命

当前阶段，人工智能从“单纯对话”走向“多步深度思考”。OpenAI o1/o3 与开源黑马 DeepSeek-R1 将重心带回推理时计算（Test-Time Compute），AI 能够像人一样深思熟虑；同时，统一协议 MCP 的普及推动了全自动 AI Agent 的大范围普及，具身智能和物理 AI 也迎来了真正的落地时代。

2024-02-15 模型

OpenAI 发布 Sora 文本生成视频模型

OpenAI 推出 Sora，能够根据文字生成 60 秒极具景深、动态连贯以及理解物理世界逻辑的视频片段，将 AIGC 领域的竞争从图文拉入高精度视频生成赛道。

2024-05-13 模型

GPT-4o：原生端到端实时多模态交互

OpenAI 推出 GPT-4o，模型统一在一个神经网络上处理音频、视频和文本输入，实现低延迟（232 毫秒）的实时语音通话，彻底重塑了人机交互的自然体验。

2024-09-12 技术/算法

OpenAI o1 开启推理时计算（Test-Time Compute）

OpenAI 发布 o1 系列推理模型。引入内置思考链（Chain of Thought），通过强化学习在输出前多路径思考。标志着大模型从“简单词语拟合（直觉）”过渡到“复杂逻辑思辨（慢思考）”的算力演化路线。

2024-11-25 框架/工具

Anthropic 发布 Model Context Protocol (MCP) 统一框架

Anthropic 开源 MCP 模型上下文协议，为 LLM 连接各种开发工具、数据库和 API 提供了标准互联接口，直接打通了 Agentic AI 落地的信息孤岛生态。

2025-01-20 公司/事件

DeepSeek-R1 掀起全球大模型效率海啸

中国初创公司 DeepSeek 开源推理模型 R1，以极小的算力和极其创新的 GRPO 算法实现了可匹敌 OpenAI o1 的卓越推理性能，在全球科技市场引发对于高资金算力垄断神话的破灭，推动了推理效率平权运动。

2025-04-15 框架/工具

MCP 移交 Linux 基金会，正式确立 Agent 标准

为推动开源大一统，MCP 协议托管至 Linux 基金会，在 2025 年底突破数千万部署，使得“自主规划、自我纠错”的 AI Agent 真正走向企业级协同作业主流。

2026-03-16 技术/算法

NVIDIA Blackwell 架构全面量产部署

黄仁勋在 GTC 2026 上宣布 Blackwell GPU 大规模商用，并推进 2nm 处理芯片的生产进程，在超大规模预训练硬件端进一步将单位算力功耗降低 30% 以上。

2026-05-26 公司/事件

具身智能与物理 AI 走入产业试点

随着端侧强推理模型的算法蒸馏获得突破，大模型作为“物理大脑”被快速集成至 Figure 03 与 Tesla Optimus 3 代等 humanoid 机器人，在汽车制造和仓储物流中执行自主物料装配试点。

分化与交替：壮大与凋落的技术路线

↑ 蓬勃生长的主流分支

Transformer 架构

完全颠覆了 RNN，以自注意力机制实现极致的并行化与长上下文建模，如今已一统 CV (ViT) 和 NLP (LLM) 底层架构。

Diffusion 扩散模型

替代了难以训练的 GAN，生成多样性高、训练极度稳定，并极易与 CLIP 语义关联，主宰了从 Stable Diffusion 到 Sora 的图像与视频领域。

RL 推理与思考链 (Test-Time Compute)

超越了预训练拟合海量文本的局限，允许模型在推理时消耗算力进行“多步推导与逻辑纠错”，成为打破智能天花板的新路线。

Agentic AI (智能体生态)

从单纯的 Chatbot（单问单答）进化为具备自主调取 API、规划目标、分工协作与自我验证的闭环自动化系统。

↓ 逐渐式微/转型的分支

RNN / LSTM 循环网络

由于长序列依赖的“梯度消失”和天生“无法并行训练”的串行计算缺陷，在 2018 年 BERT 和 GPT 兴起后迅速退居二线。

GAN 生成对抗网络

训练极易发生模式崩溃且无法精准对齐复杂文本 Prompt。2022 年后除特定实时图像增强或超分场景外，基本被 Diffusion 全面取代。

单一监督微调 (SFT)

对每一项小任务分别搜集海量数据集并微调小模型的方法基本被放弃，被基于大模型的 zero-shot/few-shot In-Context 提示及 Agent 代替。

独立 AI 硬件 (Standalone AI Devices)

如 Humane AI Pin 和 Rabbit R1 曾试图颠覆智能手机，但因延迟过高、交互痛点和续航局限被边缘化，硬件市场转向本地 OS 网关（Copilot, Apple Intelligence）与具身/AR 眼镜形态。
